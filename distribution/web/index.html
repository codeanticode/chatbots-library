Download

Download chatbots version 0.1 in .zip format.

Installation

Unzip and put the extracted chatbots folder into the libraries folder of your processing sketches. Reference and examples are included in the chatbots folder.

Keywords chatbots,ai,natural language

Reference. Have a look at the javadoc reference here. a copy of the reference is included in the .zip as well.

Source. The source code of chatbots is available at github, and its repository can be browsed here.
